# screenshot-folder-guard
Kids on a computer. They learn fast, but they mess things up constantly, so you can’t stop watching them.
A friend is trying to teach them how to use the Windows Snipping Tool. This tool normally saves snips to a default folder. Everything goes fine until one of the kids moves the entire folder instead of just the screenshot.
This program prevents the folder from being moved by keeping a hidden file open.
It includes a simple installer and doesn’t require administrator privileges.
